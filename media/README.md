# Медиа файлы проекта

Эта папка содержит все медиа-файлы, используемые ботом.

## Изображения

### Основные
- `start.png` - Приветственное изображение при /start
- `veo3.png` - Изображение процесса генерации
- `statistic.jpeg` - Статистика для мемов

### Мемы
- `mops.jpeg` / `mops.jpg` - Изображение мопса для мема
- `2568-11-12_16.23.25-removebg-preview.png` - Обработанное изображение

## Видео

### Примеры мемов
- `mother.MP4` - Видео для мема "Мама такси"
- `mopsvideo.mp4` - Видео мопса

## Использование

Эти файлы используются ботом для:
- Приветственных сообщений
- Показа процесса генерации
- Примеров мемов
- Статистики

## Пути в коде

При перемещении файлов обновлены пути в следующих местах:
- `src/bot_start.js` - приветственное изображение и процесс генерации
- `src/utils/memeLoader.js` - загрузка примеров мемов
